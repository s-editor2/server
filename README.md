# S-Editor Server
Script to run Smart Editor offline server locally on your device.

## Install on Linux (Ubuntu & Debian)

- ### Updating
  - ` sudo apt-get update`
  - ` sudo apt-get upgrade -y`

- ### Installing The Requirements
  - ` sudo apt-get install curl python -y `

- ### Installing CLI
  - ` curl  ##### | bash `
